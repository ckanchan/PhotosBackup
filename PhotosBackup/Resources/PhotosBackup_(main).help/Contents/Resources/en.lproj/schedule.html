Scheduling backups

Photos Backup can run as a background process and schedule backups to occur hourly, daily or weekly. This can be easily set in the Schedule pane:

When the schedule is activated, Photos Backup will deliver notifications in the background regarding whether backups are successful or not. If a backup cannot be completed for some reason, Photos Backup will notify you and ask macOS to defer it to a later time and try again if possible. The menu bar icon will also change to reflect the app's current status:

Idle
Back up in progress
An error has occurred
