Sparse Bundles

The sparsebundle format is a disk image format created by Apple. It can be stored on most file systems, and can grow in size up to a maximum size specified at creation. Thus, a 10GB sparsebundle will initially take up far less space on disk. The sparsebundle itself takes the form of an Apple package - a folder that appears as a single file on Mac systems (like applications do). It will appear as a folder on non-Apple operating systems - do not modify its internals. In order to access the Photos Library backed up within, you must mount the sparsebundle as a disk using the Finder or other Mac utilities.

See the macOS Disk Utility manual for more details.
