Setup

On first run Photos Backup will need to be set up with a photos library, backup location and optionally a schedule on which to back up regularly.

On macOS 10.14 you may need to grant Photos Backup access to your Photos Library in Security Preferences due to improved privacy protection. Photos Backup will automatically prompt you for this. You can revoke this permission at any time from System Preferences, but then Photos Backup will stop working! However, if your photos library isn't located under ~/Pictures/ (where ~ is your Home folder) then the macOS privacy protections do not apply.

After running the application for the first time, the Configuration panel will pop up automatically:

1 Pane switcher Access backup settings, the schedule, debug log or the About screen.
2 Photos Library settings Set the location of your photos library here, either by clicking on Set to use the file browser or entering the path directly. Photos Backup will calculate the size automatically
3 Backup Location settings Manages the sparsebundle file that will be the backup location. Set or create a new backup location here
4 Status Quick overview of any problems. If everything is set up correctly, click on Back Up Now to initiate a backup
5 Last backup Shows the date and time of the most recent backup performed by Photos Backup
6 Back Up Now When everything is set up correctly, click on this button to start a back up.

Setting the location of the photos library

To set the location of the photos library you would like to back up, either click the Set button to open the file picker, or type the path into the location field directly and press RETURN.

Setting up a new backup location

Photos Backup uses the sparsebundle disk image format to store its back ups. In most cases, you will want Photos Backup to create a new backup location.

To create a new backup location click Create new backup location... which will open the file picker panel. Select the folder in which you want to store your backups (which can be on a local, removable or remote drive), and click Create. Photos Backup will then create the sparsebundle and provision it with a capacity 1.5x the size of the selected photos library.

If you have not selected a photos library then Photos Backup will automatically create a 2GB sparsebundle. It is strongly recommended you select your photos library first so that you can be certain the backup location will have enough storage space.

Setting up an existing backup location

You can set a pre-existing sparsebundle instead of creating a new one. This is useful if you have used Photos Backup before, or are an advanced user who has created a sparsebundle manually. The backup location must be a sparsebundle with the name photosBackup. To select a previously existing backup location, click Set, which will open the file picker panel.

Once fully set up, you can click Back Up Now to run an immediate backup, or configure the schedule and let Photos Backup automatically run in the background.
